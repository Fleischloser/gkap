Anleitung anhand Eclipse IDE

Neues Project anlegen
Rechtsklick im Package Explorer -> New Java Project (mit JavaSE 1.7) -> Finish

Importieren
Rechtsklick auf der src-Folder im neu erstellten Projekt -> Import -> General -> Archive File -> Next -> Jar-Datei auswählen -> "Into Folder" sollte auf den src-Folder zeigen -> Finish 

Ausführen
tests/AIGraphTest.java ausführen
