Anleitung anhand Eclipse IDE

Neues Project anlegen
Rechtsklick im Package Explorer -> New Java Project (mit JavaSE 1.7) -> Finish

Importieren
Rechtsklick auf der src-Folder im neu erstellten Projekt -> Import -> General -> Archive File -> Next -> ZIP-Datei auswählen -> "Into Folder" sollte auf den src-Folder zeigen -> Finish

tests/FordFulkersonEdmondKarpTest.java starten mit TRUE als useFord Pararmeter
